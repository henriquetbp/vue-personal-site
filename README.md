
# Vue Personal Website

  

Projeto de site pessoal SPA utilizando Vue.js.

  

## Versão Utilizada

  

Vue.js 3

  

## Instruções

  

Clone este repositório
```sh

git clone https://github.com/henriquetbp/vue-personal-site.git

```
Instale as dependências
```sh

npm  install

```
Execute o projeto
```sh

npm  run dev

```
